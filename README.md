# Vita AI Assistant 🏥🤖

Vita AI Assistant es un asistente inteligente basado en IA que permite responder preguntas en lenguaje natural sobre la documentación de una clínica ficticia. El proyecto utiliza técnicas de **Retrieval-Augmented Generation (RAG)** para buscar información relevante en documentos y generar respuestas precisas.

## Objetivo

Facilitar el acceso a la información de la clínica mediante un asistente capaz de responder consultas sobre políticas, convenios, citas médicas y otros documentos internos.

## Tecnologías

* Python
* LangChain
* Google Gemini
* ...
* Oracle Cloud Infrastructure (OCI)

## Funcionalidades

* Carga y procesamiento de documentos PDF.
* Búsqueda semántica mediante embeddings.
* Respuestas en lenguaje natural utilizando un modelo de IA.
* Interfaz web para realizar consultas.
* Despliegue en Oracle Cloud.
